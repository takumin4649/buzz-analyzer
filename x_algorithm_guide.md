## 1. アルゴリズムの全体像

Xの「For You」フィードは2段階で動く：

### STEP 1: 候補集め（Candidate Retrieval）
- フォロー中（In-Network）から750件
- 似た興味の非フォロー（Out-of-Network）から750件
- 合計1,500件をピックアップ

### STEP 2: ランキング（Phoenixスコアラー）
- GrokベースのTransformerモデル（xAIが開発）
- 1投稿ごとに「いいね・リプ・RTする確率」を全部予測
- その確率に重みを掛けて最終スコアを計算 → 上位を表示
- 毎日1億件以上の投稿を全文読んで評価（Elon本人が2025/10に発表）
- 2025年10月〜11月に「ヒューリスティック全削除 → Grok完全移行」

### STEP 3: フィルタリング
- 既読投稿、ミュート・ブロック対象を除外
- 多様性フィルター（同一ユーザー連続表示の抑制）

## 2. エンゲージメント重み（公式コード解析値）

### スコア計算式
```
Final Score = Σ (weight_i × P(action_i))
```

### 重みテーブル（いいね=0.5基準）

| アクション | 重み | いいね比 | 意味 |
|-----------|------|---------|------|
| リプへの返信（著者自身） | **75.0** | **150倍** | 著者がリプに返信するだけで最高評価 |
| リプライ | **13.5** | **27倍** | いいねの27倍の価値 |
| プロフィールクリック | **12.0** | **24倍** | 興味を持たれている証拠 |
| 会話クリック（スレッド展開） | **11.0** | **22倍** | スレッドが展開される = 高品質 |
| ブックマーク | **10.0** | **20倍** | 「後で読む」= 価値が高い |
| RT（リポスト） | **1.0** | **2倍** | 拡散行動 |
| いいね | **0.5** | **1倍** | 基準値 |
| ネガティブ（非表示等） | **-74.0** | **-148倍** | 1件で大ダメージ |
| 通報 | **-369.0** | **-738倍** | 1件でほぼ終了 |

### 超重要ポイント
- **「著者自身がリプに返信する」だけで150倍** → これが一番バズるトリガー
- **いいねはほぼ無意味**（RT・ブックマーク・リプが命）
- **ネガティブシグナルはとんでもなく重い**（通報1件でほぼ死）

## 3. 2025年以降のGrok導入による変化

### 3.1 Grok AIによるコンテンツ評価
- Grokが毎日1億件以上の投稿を**全文読む**
- **トーン分析が本格化**: ポジティブ・建設的 → 拡散促進、攻撃的・rage bait → 抑制
- AI生成コンテンツの検知を厳しく、有機的コンテンツを優先

### 3.2 コンテンツタイプ別の影響
| 要素 | 影響 |
|------|------|
| **テキスト投稿** | 動画より**30%強い**（他SNSと完全に逆） |
| **スレッド** | 単発の**3倍以上**エンゲージメント |
| **外部リンク** | **30-50%リーチ減**（非Premiumはほぼゼロ） |
| **X Premium** | **4倍ブースト**（In-Network 4x、Out-of-Network 2x） |

### 3.3 滞在時間（Dwell Time）
- Xが**最重視**する指標の一つ
- 2分以上の滞在で **+10.0の重み**
- ユーザー滞在時間 = 広告接触 = X収益 → だから最重要

### 3.4 早期エンゲージメント
- **投稿後1時間以内**で全体スコアの**50%以上**が決まる
- 最初の30分のエンゲージメントが特に重要
- → ゴールデンタイム（18-21時）に投稿する理由

### 3.5 フォロワー比率ペナルティ
- フォロー500以上 AND フォロー/フォロワー比 > 0.6 でペナルティ発動
- ペナルティ: `score / exp(5 × (ratio - 0.6))`
- → **フォロワーよりフォロー数が多いとスコアが激減**

## 5. 拓巳型バズ戦略 × アルゴリズム最適化

### 既存の強み（アルゴリズム的に正しい理由）
| 拓巳の戦略 | なぜアルゴリズムに効くか |
|-----------|---------------------|
| 等身大の告白 | Grokがポジティブ・建設的と評価 → 拡散促進 |
| 具体的体験 | 滞在時間が長くなる → dwell weight加算 |
| 18-21時投稿 | 早期エンゲージメント最大化 → 最初の1時間で勝負 |
| CTAなしでも伸びる | Grokは自然な会話を評価、押し付けCTAは逆効果の場合も |
| 秘匿感フレーズ | プロフィールクリック誘発（24倍の重み）|
| 「ド素人」「正直」系 | 共感リプライ誘発（27倍の重み）|

### アルゴリズムに基づく追加施策
1. **リプライに必ず返信する**（150倍ブースト）
2. **スレッド形式を試す**（3倍エンゲージメント）
3. **外部リンクは本文に入れない**（リプに書く）
4. **ブックマーク誘発フレーズ**（「保存推奨」→ 20倍の重み）
5. **長文テキスト投稿**（動画より30%強い）
